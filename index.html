DevGib

A Chrome extension that analyzes the quality of programming-related links.

Install

from Chrome Web Store

Source

on Github

DevGib is a Chrome extension that analyzes the quality of programming-related resources before viewing them, and shows the ranking next to their link:

DevGib currently supports:
  • Github repositories.
  • StackOverflow questions.


DevGib works anywhere in the web.

How to setup DevGib:
  • Install the extension.
Optional setup (for unlimited Github ranking):
  • After installation, click the DevGib extension icon, and press the Generate access token link. In the Github application page, press Generate new token. Use the default scopes, and press Generate Token.
  • Copy the generated token to the text field inside the DevGib extension popup.
